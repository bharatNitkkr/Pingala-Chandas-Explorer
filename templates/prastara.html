Prastāra Order

The lecture-style order starts with all 0s, meaning all Guru. Then the leftmost bit changes fastest.

For n = 3, the order becomes:

000, 100, 010, 110, 001, 101, 011, 111

This is the standard binary sequence when read from right to left.
