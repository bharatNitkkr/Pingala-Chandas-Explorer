Sūtra Details from Piṅgala's Chandaḥsūtra

This section connects each algorithm in the application with the corresponding sūtras from Piṅgala's Chandaḥsūtra. The text is written in compact sūtra style, so the algorithmic meaning is traditionally understood with the help of commentary, especially Halāyudha's commentary.

1. Prastāra Algorithm

Purpose: To enumerate all possible Laghu-Guru patterns for a given length.

Original Sūtras

द्विको ग्लौ ॥ ८.२० ॥
मिश्री च ॥ ८.२१ ॥
पृथग्ला मिश्राः ॥ ८.२२ ॥
वसवस्त्रिकाः ॥ ८.२३ ॥

Padaccheda

द्विकः + ग्लौ
मिश्री + च
पृथक् + लाः + मिश्राः
वसवः + त्रिकाः

Word Meaning

Term Meaning
ग् Guru
ल् Laghu
मिश्री Mixed / combined arrangement
पृथक् Separately
त्रिकाः Groups of three / triples

Algorithmic Interpretation

Prastāra expands a given length into all possible arrangements of Guru and Laghu. In the lecture-style order, for n = 3, the patterns are:

000 100 010 110 001 101 011 111

Here 0 = Guru and 1 = Laghu.

2. Naṣṭa Algorithm

Purpose: Given a rank number, recover the corresponding Laghu-Guru pattern.

Original Sūtras

लर्धे ॥ ८.२४ ॥
सैके ग् ॥ ८.२५ ॥

Padaccheda

ल् + अर्धे
स + एके + ग्

Word Meaning

Term Meaning
ल् Laghu
अर्धे On halving / when divided by two
सैके With one added
ग् Guru

Algorithmic Interpretation

Start with rank q = r. If q is even, write 1 / Laghu and set q = q / 2. If q is odd, write 0 / Guru and set q = (q + 1) / 2. Continue until n bits are obtained.

Example

Rank r = 5, length n = 3:

  • 5 is odd → write 0 → q = (5 + 1) / 2 = 3
  • 3 is odd → write 0 → q = (3 + 1) / 2 = 2
  • 2 is even → write 1 → q = 2 / 2 = 1

Pattern: 001 = Guru Guru Laghu

3. Uddiṣṭa Algorithm

Purpose: Given a Laghu-Guru pattern, find its rank in its length class.

Original Sūtras

प्रतिलोमगणं द्विः लाद्यम् ॥ ८.२६ ॥
ततो गि एकं जह्यात् ॥ ८.२७ ॥

Padaccheda

प्रतिलोम-गणम् + द्विः + ल्-आद्यम्
ततः + गि + एकम् + जह्यात्

Word Meaning

Term Meaning
प्रतिलोम Reverse order
गणम् Counting / calculation
द्विः Twice
ल्-आद्यम् Beginning with Laghu
गि When Guru occurs
एकम् जह्यात् Subtract one

Algorithmic Interpretation

Start with r = 1. Read the pattern from right to left.

  • If the bit is 1 / Laghu, then r = 2r.
  • If the bit is 0 / Guru, then r = 2r - 1.

Example

Pattern p = 001

  • Start r = 1
  • Read 1 → r = 2 × 1 = 2
  • Read 0 → r = 2 × 2 - 1 = 3
  • Read 0 → r = 2 × 3 - 1 = 5

Final rank = 5.

4. Saṁkhyāna Algorithm

Purpose: To find the total number of patterns for a given length n.

Original Sūtras

द्विः अर्धे ॥ ८.२८ ॥
रूपे शून्यम् ॥ ८.२९ ॥
द्विः शून्ये ॥ ८.३० ॥
तावदर्धे तद्गुणितम् ॥ ८.३१ ॥

Padaccheda

द्विः + अर्धे
रूपे + शून्यम्
द्विः + शून्ये
तावत् + अर्धे + तत् + गुणितम्

Word Meaning

Term Meaning
द्विः Two / twice
अर्धे On halving
रूपे When one remains
शून्यम् Zero
गुणितम् Multiplied

Algorithmic Interpretation

Saṁkhyāna calculates the total number of patterns: 2n. The lecture explains this through a repeated-squaring style calculation.

Example

For n = 10:

  • 10 / 2 = 5
  • 5 - 1 = 4
  • 4 / 2 = 2
  • 2 / 2 = 1
  • 1 - 1 = 0

Reconstructing gives: 210 = 1024.

5. Adhvayōga Algorithm

Purpose: To count all patterns up to a given length n.

Original Sūtra

द्विद्यूनं तदन्तानाम् ॥ ८.३२ ॥

Padaccheda

द्विः + द्वि-ऊनम् + तत्-अन्तानाम्

Word Meaning

Term Meaning
द्विः Twice / power of two
ऊनम् Less / reduced by
तदन्तानाम् Of those ending at that point / up to that length

Algorithmic Interpretation

Adhvayōga calculates:

21 + 22 + ... + 2n = 2n + 1 - 2

6. Lagakriyā / Meru-Prastāra

Purpose: To find how many patterns of length n contain exactly 0 Guru, 1 Guru, 2 Guru, and so on.

Original Sūtras

परे पूर्णम् ॥ ८.३४ ॥
परे पूर्णम् इति ॥ ८.३५ ॥

Padaccheda

परे + पूर्णम्
परे + पूर्णम् + इति

Word Meaning

Term Meaning
परे At the end / beyond / next position
पूर्णम् Filled / complete
इति Thus

Algorithmic Interpretation

These compact sūtras are interpreted through Halāyudha's commentary as the construction of Meru-Prastāra, corresponding to Pascal's Triangle.

For n = 4:

1 4 6 4 1

Meaning:

  • 0 Guru → 1 pattern
  • 1 Guru → 4 patterns
  • 2 Guru → 6 patterns
  • 3 Guru → 4 patterns
  • 4 Guru → 1 pattern
