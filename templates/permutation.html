Permutation Algorithm

This algorithm counts and generates arrangements of n distinct objects.

Back to Other Algorithms

If there are n distinct objects, then the number of possible arrangements is:

n!

Lecture Example

In the lecture, the example discussed is Viṣṇu with four hands and four āstras:

Śaṅkha Cakra Gadā Padma

The number of possible arrangements is:

4! = 24
