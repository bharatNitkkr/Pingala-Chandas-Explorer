Lagakriyā / Meru-Prastāra Algorithm

Lagakriyā answers this question: for a Chandas pattern of length n, how many patterns contain exactly 0 Guru, 1 Guru, 2 Guru, and so on?

This is represented using Meru-Prastāra, known today as Pascal's Triangle.
