Pingala Chandas Explorer

An interactive full-stack web application to understand the six algorithms of Pingala’s Chandaḥśāstra.

This app demonstrates Sanskrit Chandas using modern computer science ideas like binary numbers, recursion, combinatorics, indexing, and Pascal Triangle.
