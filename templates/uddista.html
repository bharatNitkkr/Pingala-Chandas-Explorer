Uddiṣṭa Algorithm 1

Start with r = 1. Proceed from right to left.

  • If the bit is 1, then new r = 2r.
  • If the bit is 0, then new r = 2r - 1.

Uddiṣṭa Algorithm 2

This is the modern binary-to-decimal style method shown in the lecture. Write 1, 2, 4, 8, ... under the bits from left to right. Discard the weights under 0, add the remaining weights, and finally add 1.
