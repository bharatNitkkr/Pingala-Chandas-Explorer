About Pingala Chandas Explorer

Pingala Chandas Explorer is an educational web application that connects Sanskrit Chandas, ancient Indian combinatorics, binary representation, and modern computer science.

Project Overview

This project demonstrates the mathematical ideas found in Piṅgala's Chandaḥsūtra through interactive algorithms and visual explanations. The main goal is to help learners understand how Laghu-Guru patterns in Sanskrit prosody naturally lead to binary sequences, counting methods, ranking, unranking, and combinatorial structures.

The application was developed as a working educational prototype for studying the relationship between Sanskrit Chandas and computational thinking.

Inspiration

This project is inspired by the lecture:

The Art and Science of Sanskrit Chandas: Origins of Binary System and Combinatorics

Speaker: Prof. Arnab Bhattacharya
Department of Computer Science and Engineering
IIT Kanpur

The lecture explains how the Laghu-Guru structure of Sanskrit Chandas can be studied using binary representation and combinatorial algorithms.

Algorithms Covered

Algorithm Purpose Modern Interpretation
Prastāra Enumerates all possible Laghu-Guru patterns of length n Binary pattern generation
Naṣṭa Finds the Laghu-Guru pattern from a given rank Unranking algorithm
Uddiṣṭa Finds the rank of a given Laghu-Guru pattern Ranking algorithm
Saṁkhyāna Counts total patterns for a given length Calculation of 2n using repeated squaring
Adhvayōga Counts all patterns up to a given length n Geometric series: 21 + 22 + ... + 2n
Lagakriyā / Meru-Prastāra Counts patterns with exactly 0 Guru, 1 Guru, 2 Guru, etc. Binomial coefficients and Pascal's Triangle
Mātrā Chandaḥ Counts patterns by total mātrā value instead of fixed akṣara length Fibonacci-style recurrence: M(m) = M(m - 1) + M(m - 2)

Key Concepts

  • Laghu: A short syllable, represented as 1 in this app.
  • Guru: A long syllable, represented as 0 in this app.
  • Chandas: The study of poetic meter in Sanskrit.
  • Prastāra: Expansion of all possible patterns.
  • Meru: A triangular arrangement corresponding to binomial coefficients.
  • Binary Representation: A two-symbol system that connects Laghu and Guru with modern computation.

Sūtra-Based Study

The project also includes a dedicated Sūtra Details section. This section connects each algorithm with the corresponding sūtras from Piṅgala's Chandaḥsūtra and provides:

  • Original Sanskrit sūtra text
  • Padaccheda
  • Word-by-word meaning
  • Algorithmic interpretation
  • Examples based on the lecture method

Educational Purpose

This application is designed for students, researchers, and learners who want to explore the mathematical structure of Sanskrit Chandas in an interactive way.

It is not intended to replace traditional study of Chandaḥśāstra, but to support it by showing how the algorithms can be understood through computation and visualization.
